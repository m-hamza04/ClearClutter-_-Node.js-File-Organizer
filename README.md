# 🧹 ClearClutter

A simple Node.js script to automatically organize messy folders by sorting files into directories based on their file extensions.

---

## 🚀 Features

- 📂 Automatically detects file types
- 🗂️ Creates folders for each extension (jpg, png, pdf, etc.)
- 🔄 Moves files into their respective folders
- ⚡ Fast and lightweight
- ❌ Skips unwanted files (like `.json`)
